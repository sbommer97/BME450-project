## Project description
I enjoy bass fishing so I figured a classifier would be fun to make. There are generally considered to be 3 main species of black bass in north America (largemouth, smallmouth and spotted) and some people don’t know the difference between them. My one idea for a dataset is getting ahold of a kayak tournament group since with some of them they record entries by taking pictures of each fish and do have standards for admission that I think could help with image quality, and I imagine a large organization would easily have thousands of files. If that doesn’t work, I can source from the internet. I see a few progressive goals that would work for this project. First a bare minimum being largemouth vs smallmouth detection as they have the most distinct difference, with the other part being adding the spotted bass, though I see some issues of them being less common reducing the dataset size, along with the differences between them and largemouth to be relatively small. A stretch goal I see if the tournament dataset works out would be to auto calculate length since the images in that case would have a scale for that in the background.
